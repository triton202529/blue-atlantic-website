# Blue Atlantic Website

Public corporate website for Blue Atlantic.

Official production domain:

```text
https://blueatlantic.co
```

Official contact emails:

```text
contact@blueatlantic.co
akim@blueatlantic.co
```

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Copy `.env.example` and set:

```bash
NEXT_PUBLIC_SITE_URL=https://blueatlantic.co
```

## Scripts

```bash
npm run lint
npm run build
```
